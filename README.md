# YoFood — Генератор корпоративного меню

Веб-утилита для автоматизации подготовки еженедельной таблицы корпоративного питания.

Кейтеринг присылает меню на сербском языке (.txt / .docx) — сервис переводит его на русский через Claude API, структурирует и вставляет в Excel-шаблон. На выходе — готовый .xlsx для загрузки в Google Sheets.

## Стек

- **Vue 3** + **TypeScript** + **Vite**
- **Ant Design Vue** — UI-компоненты
- **SheetJS (xlsx)** — чтение и запись .xlsx в браузере
- **Claude API** — перевод и структурирование меню (модель `claude-sonnet-4-6`)

## Как это работает

1. Пользователь вводит свой Claude API-ключ (сохраняется в `localStorage`)
2. Загружает файл меню от кейтеринга (.txt / .docx)
3. Claude API переводит меню (серб. → рус.) и возвращает структурированный JSON
4. Пользователь загружает Excel-шаблон с именами сотрудников
5. Нажимает «Сгенерировать» — получает готовый .xlsx с заполненными блюдами

## Безопасность и приватность

- **Нет серверной части** — всё работает в браузере
- **API-ключ хранится только в `localStorage`** — не отправляется никуда, кроме `api.anthropic.com`
- **Никакие данные не сохраняются на сервере** — файлы обрабатываются исключительно в памяти браузера
- **Имена сотрудников не покидают браузер** — Excel-шаблон с именами обрабатывается локально и не отправляется ни на сервер, ни в Claude API
- **Нет базы данных, нет аккаунтов, нет аналитики**
- Проект не хранит и не передаёт секретные данные третьим сторонам
