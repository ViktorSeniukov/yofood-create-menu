Privacy Policy

Last updated: May 2, 2026

1. Overview

Menu Generator ("Add-on") is a Google Sheets Add-on designed for internal use. This policy explains what data is handled and how.

2. Data We Collect

The Add-on does not collect, store, or transmit any personal data to our servers. There are no servers.

3. Data Handled Locally

  • Anthropic API key — stored in Google Apps Script User Properties, accessible only to you within your Google account. It is never sent anywhere except to the Anthropic API.
  • Menu file contents — sent directly from your browser to the Anthropic Claude API for translation. Not stored anywhere.
  • Spreadsheet data — the Add-on writes translated menu data into the currently active spreadsheet. No spreadsheet data is read beyond what is necessary to locate the target cells.

4. Third-Party Services

Menu file contents are sent to the Anthropic Claude API for processing. Anthropic's data handling is governed by their Privacy Policy.

No other third-party services receive any data.

5. Google API Scopes Used

  • spreadsheets.currentonly — read/write the active spreadsheet only
  • script.external_request — send HTTP requests to the Anthropic API
  • script.container.ui — display the sidebar UI
  • userinfo.email — identify the current user for storing their API key

The Add-on's use of Google user data complies with the Google API Services User Data Policy, including the Limited Use requirements.

6. Data Retention

No user data is retained by the Add-on beyond the current session, except for the API key stored in Google User Properties (which you can delete at any time from within the Add-on).
