# UI/UX Спецификация: Menu Generator

## Стек
Vue 3 + Ant Design Vue. Один SPA-экран, без роутинга.

---

## Лейаут страницы

Страница **не скроллится целиком** — занимает ровно `100vh`.

```
App (height: 100vh, display: flex, flex-direction: column, overflow: hidden)
├── AppHeader       (height: 48px, flex-shrink: 0)
└── AppMain         (flex: 1, overflow: hidden, display: flex, gap: 16px, padding: 16px)
    ├── LeftColumn  (flex: 1, min-width: 0, display: flex, flex-direction: column, gap: 12px)
    └── RightColumn (width: 272px, flex-shrink: 0, display: flex, flex-direction: column, gap: 12px)
```

---

## AppHeader

Высота 48px, белый фон, `border-bottom` стандартный токен Ant Design.  
Внутри — flex, `justify-content: space-between`, `align-items: center`, `padding: 0 20px`.

**Слева:** текст «Меню-генератор», `font-size: 14px`, `font-weight: 500`.

**Справа:** кнопка-пилюля «API-ключ».  
Форма: `border-radius: 20px`, `border: 1px solid` (токен Ant border), `padding: 4px 10px`, `font-size: 12px`.  
Внутри — цветная точка (`width: 7px, height: 7px, border-radius: 50%`) + текст «API-ключ»:
- Точка **зелёная** — если API-ключ сохранён
- Точка **красная** — если ключ не задан

По клику на кнопку — открывается `ApiKeyDrawer`.

---

## ApiKeyDrawer

`a-drawer`, `placement="right"`, `width: 300px`.

**Шапка drawer:** заголовок «Настройки API» + кнопка закрытия ×.

**Тело:**
- Лейбл «Claude API ключ» (`font-size: 12px`, приглушённый цвет)
- `a-input` `type="password"`, placeholder `sk-ant-api03-...`
- Подпись под инпутом: «Ключ сохраняется только в localStorage вашего браузера и никуда не отправляется», `font-size: 11px`, приглушённый цвет
- Если ключ уже сохранён — под подписью показывается зелёный блок-статус: иконка галочки + текст «Ключ сохранён и активен», `background: colorSuccessBg` (токен Ant), `border-radius: 8px`, `padding: 10px 12px`

**Подвал drawer** (`border-top`):  
Кнопка «Сохранить» — `a-button`, `type="primary"`, `block`.

## LeftColumn

### Карточка 1 — Меню от кейтеринга

`a-card`, `flex-shrink: 0`.  
Заголовок карточки: «Меню от кейтеринга».

#### Состояние: файл не загружен

`a-upload` с `type="dragger"` на всю карточку. Принимает `.txt` и `.docx`.

Содержимое dragger-зоны (по центру, flex column, gap: 8px):
- Иконка загрузки (стрелка вверх) в квадрате `30×30px` на фоне `colorBgLayout`
- Текст: **«Перетащите файл»** (жирный) + « или нажмите», `font-size: 12px`
- Подпись: «.txt или .docx», `font-size: 11px`, приглушённый

Пока идёт запрос к Claude после загрузки файла — поверх карточки показывается `a-spin` (оверлей).

#### Состояние: файл загружен, идёт запрос

Dragger скрывается. Вместо него — строка с файлом:
- Иконка галочки в квадрате `30×30px`, `background: colorSuccessBg`
- Имя файла, `font-size: 13px`, `font-weight: 500`
- Под именем: «Загружено · [N] дней · [M] блюд», `font-size: 11px`, приглушённый
- Кнопка × сброса справа (иконка, без рамки, `opacity: 0.35`, при hover `opacity: 1`)

Вся строка: `background: colorBgLayout`, `border-radius: 8px`, `padding: 10px 12px`.

#### Состояние: ошибка запроса к Claude

Под строкой с файлом показывается `a-alert` `type="error"` с текстом ошибки.  
Кнопка × сброса остаётся активной — пользователь может сбросить и загрузить снова.

### Карточка 2 — Результат перевода

`a-card`, **`flex: 1`**, **`min-height: 0`**, **`overflow: hidden`**.

Карточка занимает весь оставшийся вертикальный размер колонки и **не растягивает страницу**.

#### Состояние: меню ещё не загружено

Пустой placeholder по центру карточки:
- Иконка (например, `FileTextOutlined`), `font-size: 32px`, приглушённый цвет
- Текст «Загрузите меню для отображения результата», `font-size: 13px`, приглушённый

#### Состояние: меню загружено и переведено

**Шапка карточки** (не скроллится):
- Заголовок «Результат перевода»
- Кнопка «Скачать таблицу» — `a-button`, `size="small"`, иконка стрелки вниз, справа в шапке

**Табы** (не скроллятся): `a-tabs`, `type="line"`, по одному табу на каждый день — «Пн», «Вт», «Ср», «Чт», «Пт». Показываются только дни, которые есть в данных.

**Тело таба** — **скроллится вертикально**, `overflow-y: auto`, высота ограничена оставшимся пространством карточки.

Внутри таба — секции по категориям (Сок, Завтрак, Суп, Горячее, Гарнир, Салат, Десерт). Порядок фиксированный, пустые категории не показываются.

Каждая секция:
```
section-head (padding: 8px 14px 6px)
├── Бейдж с названием категории
│   background: colorBgLayout, border-radius: 10px
│   font-size: 10px, font-weight: 500, приглушённый цвет
│   padding: 2px 8px
└── Счётчик «N позиций», font-size: 11px, приглушённый

section-items (padding: 0 14px 10px)
└── dish (border-bottom между позициями, padding: 5px 0)
    ├── Русское название, font-size: 12px, основной цвет, line-height: 1.45
    └── Оригинал на сербской латинице, font-size: 11px, приглушённый, margin-top: 1px
```

## RightColumn

### Карточка 1 — Шаблон .xlsx

`a-card`, `flex-shrink: 0`.  
Заголовок: «Шаблон .xlsx».

#### Состояние: файл не загружен

`a-upload` `type="dragger"`. Принимает только `.xlsx`.

Содержимое dragger-зоны аналогично карточке меню:
- Иконка загрузки
- «**Перетащите файл** или нажмите»
- Подпись: «.xlsx»

#### Состояние: файл загружен

Строка с файлом — аналогичная карточке меню:
- Иконка галочки на зелёном фоне
- Имя файла + мета «Загружено»
- Кнопка × сброса

---

### Карточка 2 — Готовность

`a-card`, `flex-shrink: 0`.  
Заголовок: «Готовность».

Чеклист из двух пунктов, `display: flex, flex-direction: column, gap: 8px`:

Каждый пункт — строка `display: flex, align-items: center, gap: 8px, font-size: 12px`:

| Иконка | Текст | Условие |
|--------|-------|---------|
| Кружок `16×16px`, `background: colorSuccessBg`, галочка `stroke: colorSuccess` | «Меню переведено ([N] дней)» | JSON с меню есть в сторе |
| Кружок `16×16px`, `background: colorBgLayout`, `border: 1px dashed` | «Шаблон не загружен» / «Шаблон загружен» | В зависимости от наличия файла |

Когда пункт выполнен — иконка зелёная с галочкой, текст основного цвета.  
Когда не выполнен — иконка серая пунктирная, текст приглушённый.

---

### Блок 3 — Кнопка генерации

Не в карточке — просто в потоке колонки.

`a-button`, `type="primary"`, `block`, текст «Сгенерировать .xlsx».

**Disabled** если хотя бы одно условие из чеклиста не выполнено.

Под кнопкой — подсказка `font-size: 11px`, приглушённый, `text-align: center`:
- Если не загружено меню: «Загрузите меню для продолжения»
- Если не загружен шаблон: «Загрузите шаблон для продолжения»
- Если всё готово: подсказка скрыта

По успешной генерации — `a-message.success('Файл готов!')` и автоматическое скачивание `.xlsx`.

## Обработка ошибок

| Ситуация | Что показывать |
|----------|---------------|
| Нет API-ключа при загрузке меню | `a-message.error('Введите API-ключ в настройках')` |
| Сетевая ошибка Claude API | `a-alert type="error"` под строкой файла в карточке меню |
| Claude вернул невалидный JSON | `a-alert type="error"` «Не удалось распознать меню. Попробуйте ещё раз» |
| Ошибка генерации xlsx | `a-message.error('Ошибка генерации файла')` |

## Цвета

Хардкодить цвета запрещено. Использовать только токены Ant Design Vue:

| Токен | Где используется |
|-------|-----------------|
| `colorSuccessBg` | Фон иконки загруженного файла, фон статуса ключа, фон галочки в чеклисте |
| `colorSuccess` | Цвет галочки, цвет точки в хедере если ключ есть |
| `colorError` | Цвет точки в хедере если ключа нет |
| `colorBgLayout` | Фон строки с файлом, фон dragger, фон бейджа категории |
| `colorTextSecondary` | Все приглушённые тексты |
| `colorBorder` | Разделители между блюдами, бордеры карточек |
